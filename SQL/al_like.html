    # 1. like

    1. 功能

      like 通常用于实现模糊搜索。

    2. 语法

      select 列名 from 表名 where 列名 like 值;

      在like子句中，后边的值一般都会包含通配符，SQL 中常见的通配符如下：

      通配符 描述
      % 替代 0 个或多个字符
      _ 替代 1 个字符
      [ 字符数组 ] 字符列中的任何单一字符
      [ ! 字符数组 ] 不在字符列中的任何单一字符

    3. 例子

      我们来查询以字母 'D' 开头的所有课程信息。

      select * from courses where name like 'D%';
